API文档

文本转语音 API

POST /api/v1/tts/synthesize
请求示例 
curl -X POST "http://localhost:7280/api/v1/tts/synthesize" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "text": "你好，这是一个测试语音合成的示例",
    "provider": "openai",
    "voice": "alloy",
    "format": "mp3",
    "return_base64": false
  }' \
  --output "output.mp3"
请求参数
参数名 类型 必填 说明
text string 是 要转换的文本内容
provider string 是 服务商: "openai" 或 "gemini"
voice string 是 语音名称
format string 否 音频格式 (默认: mp3)
return_base64 boolean 否 返回Base64编码 (默认: false)
