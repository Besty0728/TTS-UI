## 快速启动
```bash
# 使用本地镜像
docker run -d \
  --name tts-ui \
  -p 7280:7280 \
  -v tts_data:/app/data \
  --restart unless-stopped \
  tts-ui:latest

# 使用docker-compose
docker-compose up -d
```
